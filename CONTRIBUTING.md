# Contributing / Come contribuire

Grazie per voler migliorare questo modulo! / Thanks for helping improve this module!

## 🇮🇹 Italiano
- **Bug / idee**: apri una *Issue* descrivendo versione PrestaShop, versione PHP, lingua e passi per riprodurre.
- **Pull Request**:
  1. Fai un fork e crea un branch (`feature/...` o `fix/...`).
  2. Mantieni lo stile del codice PrestaShop (PSR‑12 dove possibile).
  3. Non introdurre dipendenze pesanti; il modulo deve restare leggero e SEO/cache‑safe.
  4. Aggiorna `CHANGELOG.md` e, se serve, il `README.md`.
- **Aree dove serve aiuto**: revisione legale per singolo Paese UE, traduzioni, test su PrestaShop 1.7/8/9,
  ricevuta PDF, modulo Annex I‑B, log a prova di manomissione.

## 🇬🇧 English
- **Bugs / ideas**: open an *Issue* with PrestaShop version, PHP version, language and steps to reproduce.
- **Pull Requests**:
  1. Fork and create a branch (`feature/...` or `fix/...`).
  2. Follow PrestaShop code style (PSR‑12 where possible).
  3. Avoid heavy dependencies; keep the module light and SEO/cache‑safe.
  4. Update `CHANGELOG.md` and, if needed, `README.md`.
- **Help wanted**: per‑country EU legal review, translations, testing on PrestaShop 1.7/8/9, PDF receipt,
  Annex I‑B form, tamper‑evident log.

## Principi / Principles
- **Conformità prima di tutto** / compliance first (Directive (EU) 2023/2673, Art. 11a / Art. 54‑bis).
- **Nessun impatto su SEO e performance** / no SEO or performance impact.
- **Privacy by design**.
